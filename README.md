# Analisis Sentimen Komentar Pengguna Indodax

## Deskripsi
Proyek ini bertujuan untuk melakukan analisis sentimen terhadap komentar pengguna platform Indodax menggunakan Natural Language Processing (NLP).

## Metode
- Preprocessing: cleaning, tokenization, stopword removal, stemming
- Representasi teks:
  - Bag-of-Words (BoW)
  - TF-IDF
- Model: Naive Bayes

## Dataset
Dataset diambil dari Kaggle (Indodax dataset)

## Hasil
- BoW memberikan akurasi lebih tinggi dibandingkan TF-IDF
- Banyak komentar bersentimen negatif terkait biaya transaksi
